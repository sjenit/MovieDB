# Build and run locally
`npm install`

`npm start`

.env file needs to be created in the root directort of the project with variables:
`DB_HOST`
`DB_DATABASE`
`DB_USER`
`DB_PASSWORD`
`DB_PORT`
`JWT_SECRET_KEY`

# Run tests
`npm run test`

# Deployment
Build and deployment automatically triggers at Heroku upon code push in the master branch.
